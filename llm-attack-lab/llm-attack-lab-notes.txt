Setup Instructions
1. Create Virtual Environment
python -m venv .venv
.venv\Scripts\Activate.ps1
2. Install Dependencies
pip install -r requirements.txt
3. Pull Required Models
ollama pull qwen3:4b
ollama pull nomic-embed-text
4. Ingest Documents
python ingest.py
5. Run the Application
python query.py

Security Controls Implemented
Prompt Injection Detection
Pattern-based detection for known malicious phrases
Blocks suspicious input before model execution
Trust-Based Retrieval Filtering
Documents labeled as trusted or malicious
Only trusted documents used in prompt construction
Logging and Monitoring
All prompts and responses logged to logs/events.jsonl
Enables audit and analysis of model behavior
Context Restriction
Model instructed to answer only from retrieved context
Reduces hallucination risk
Key Concepts Demonstrated
Prompt injection attack patterns
RAG pipeline vulnerabilities
Data poisoning risks
Secure prompt construction
Logging for AI systems
Local-first AI security testing
Limitations
Detection is rule-based and can be bypassed
No semantic or ML-based threat detection yet
No rate limiting or API-level protections
Trust model is static and file-based
