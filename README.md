# AI Security Labs

This repository documents a four-phase AI security lab focused on building, attacking, defending, and monitoring local LLM systems.

The project starts with a basic local LLM setup and progresses into a production-style AI security pipeline with semantic guardrails, secure RAG, dynamic trust scoring, weighted retrieval, automated attack testing, and dashboard metrics.

---

## Project Goals

- Build a local LLM environment
- Create a secure RAG pipeline
- Simulate prompt injection and jailbreak attacks
- Add input and output guardrails
- Implement semantic detection using embeddings
- Add document trust scoring and weighted retrieval
- Measure security performance with automated tests and a dashboard
